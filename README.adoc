= Parkomat
image::https://github.razem.io/api/badges/razem-io/parkomat/status.svg[Build Status]

== Requirements
* Java 8
* sbt

== Configuration

In `server/conf` copy `application.sample.conf` to `application.conf` and adjust it
to your liking.

== Usage

sbt run
